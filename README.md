Version control is used to keep track of everything that is being added to a project, and to allow people to seamlessly add their work from separate workstations and from multiple developers.
Git is a tool to save your work, and GitHub is where the work is saved.
Git status shows the status of the git project, what has been added, just whats up with the repository.
Git add initially tells git that we want to add this certain thing.
Git commit -m "" is to give the git add a description so that developers know what changed. 
Git push finally uploads this information to GitHub.
